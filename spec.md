# 実装仕様書：SNS型イベントスケジュール調整アプリ

## 1. プロジェクト概要
「調整さん」のスケジュール調整機能に、「Tinder」の直感的なUIと「Instagramノート」のリアルタイムな気配（Presence）を融合させた次世代イベント管理アプリ。

### デザインコンセプト: "Vivid & Fluid"
- **遊び心:** 堅苦しさを排除し、スワイプやアニメーションを多用する。
- **直感性:** 説明不要で使えるカードUI。
- **配色:** オフホワイト (#F8F9FA) をベースに、エナジーオレンジ (#FF6B6B) とネオンパープル (#845EF7) をアクセントに使用。

## 2. 技術スタック（前提）
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS, shadcn/ui
- **Animation:** framer-motion
- **Backend (後続対応):** Supabase (Auth/DB/Storage)

## 3. 主要画面・機能詳細

### A. メインフィード画面 (`/`)
1. **みんなのノート (Instagram Notes風)**
   - 画面上部に横スクロールでユーザーアイコンを表示。
   - アイコンの上に、20文字以内の「一言メモ（吹き出し）」を表示。例：「今夜暇！」「新橋で飲み中」
2. **イベントフィード (Tinder風カード)**
   - カード形式でイベントを縦に並べる（またはスワイプ可能にする）。
   - **要素:** タイトル、主催者アイコン、日付、場所タグ。
   - **ハッシュタグ:** `#酒好きあつまれ` `#出会いを求める人向け` などをパステルカラーのバッジで表示。
   - **金額:** 一覧では非表示。
   - **ボタン:** 「詳しく見る」「検討中（Maybe）」の2つ。

### B. 初期プロフィール登録 (`/setup`)
1. **属性選択スワイプ**
   - 「アウトドア」「インドア」「深夜勢」などのカテゴリカードを左右に振って興味を登録。
2. **アレルギー/NG設定**
   - アイコン（海老、小麦、アルコール等）をタップして視覚的に設定。
3. **登録完了度バー**
   - 画面上部に進捗をパーセント表示。

### C. 人物関係（好感度）管理
- ユーザーごとに3段階のフラグを持つロジックを想定。
  - `HOT (🔥)`: 優先表示
  - `NORMAL (🍺)`: 通常表示
  - `BLOCK (🧊)`: 存在を消す（イベントを非表示にする）

### D. 招待・AIレスポンス
1. **ワンタップ回答**
   - 招待通知に対し、ボタン一つで承諾/辞退。
2. **AIレスポンス生成 (🪄)**
   - 好感度に応じた文案（熱烈な参加表明 or 角の立たない断り）を自動生成。

## 4. コンポーネント構造の指示
- `@/components/features/notes`: Instagram風ノートコンポーネント
- `@/components/features/events`: Tinder風イベントカード
- `@/components/features/profile`: プロフィール登録用パーツ
- `@/components/ui`: shadcn/uiベースの基本パーツ

## 5. AIエージェントへの実装指示（Step 1）
まずは以下の作業から開始してください。
1. `framer-motion` と `lucide-react` をインストール。
2. `app/page.tsx` にモックデータを使用した「メインフィード画面」を実装。
3. `components/features/notes` と `components/features/events` を作成し、遊び心のあるアニメーションを適用。
